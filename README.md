# Super Memory v3.0 — Reasonix 第二大脑

让 Reasonix 拥有**主动回忆、语义搜索、知识图谱**的高级记忆系统。

## 功能

| 模块 | 说明 |
|------|------|
| 🧠 会话检查点 | 每次对话结束自动摘要，下次接上 |
| 🔍 语义标签搜索 | 概念级匹配，不是关键词 |
| 📐 渐进式召回 | 摘要→要点→全文，省 token |
| 🕸️ 知识图谱 | 6 种关系类型，追溯因果链 |
| 📂 记忆分类 | 情节/语义/流程/检查点 |
| 🔀 自动巩固 | ≥5 条同主题自动合并 |
| 🪝 记忆钩子 | 消息命中已知实体自动触发 |
| ⚔️ 冲突检测 | 矛盾记忆自动提醒 |
| 📋 六种模板 | 项目/Bug/偏好/总结/模式/决策 |
| ⭐ 优先级+遗忘 | 访问越多越重要，不用自动降权 |

## 安装

```
/install-capability https://github.com/<用户名>/super-memory
```

或复制 `super-memory/` 到 `.reasonix/skills/`。
