# Super Memory v3.1 — Reasonix 第二大脑

让 Reasonix 拥有主动回忆、语义搜索、知识图谱的高级记忆系统。

## 功能

🧠 会话检查点 · 🔍 语义标签搜索 · 📐 渐进式召回 · 🕸️ 知识图谱 · 📂 记忆分类 · 🔀 自动巩固 · 🪝 记忆钩子 · ⚔️ 冲突检测 · 📋 模板 · ⭐ 优先级 · 🏥 健康检查 · ✏️ UPDATE

## 安装

```
/install-capability https://github.com/uyiuhkjuhui/super-memory
```

或复制 `super-memory/` 到 `.reasonix/skills/`。

## 版本

- v3.1: 记忆健康检查、UPDATE操作、搜索评分校准、钩子去重、初始化流程、快速参考卡
- v3.0: 会话检查点、语义标签、渐进式召回、知识图谱、记忆分类、自动巩固、记忆钩子、冲突检测
- v2.0: 主动回忆、智能关联、优先级、遗忘曲线、合并
- v1.0: 基础搜索保存 + 六种模板
